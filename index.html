? ABOUT VOXAR

VOXAR is a real-time hand-tracked AR voxel builder. Use your hands as a 3D controller to construct structures in augmented space.

TECH STACK: Three.js · MediaPipe Hands · WebGL 2.0 · Web Camera API

HAND TRACKING: Uses MediaPipe to detect 21 hand landmarks at 30fps. Pinch distance determines placement.

PHYSICS: Simulated gravity for dropped blocks when Physics Mode is enabled.
